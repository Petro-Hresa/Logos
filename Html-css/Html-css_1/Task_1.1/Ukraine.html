Назва

Етимологія словаУкраінадостименно не відома.Згідно з теорією,якої притримуються більшість українських дослідників.Украінапоходить від слів країнаабокрай,тобто уозначаєрідний,свій.Таким чиномукраїна - антонім слова чужна.[14].Згідно з однією з інших теорій, що сформувалася під впливом польської і російської історіографії, воно означаєоколицю(рос.окраину)абоприкордоння.

У попередніх століттях для позначення території України вживали слова Cкіфія,Сарматія,Військо Запоріжськетощо.

Географія
